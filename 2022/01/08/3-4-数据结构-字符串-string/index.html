3.4 字符串 string

字符串 -> 一片连续的内存空间 -> 一个由字符组成的 数组

在 C 语言中，字符串 使用 字符数组 char[] 表示；数组会占用一片连续的内存空间，内存空间存储的字节共同组成了 字符串。

ReadOnly 意味着 字符串会分配到 只读的内存空间，但是 golang 不支持直接修改 string 类型变量的内存空间；但是可以通过 string 与 []byte 类型之间的反复转换实现修改的目的：

  1. 先将这段内存拷贝到 堆 or 栈 上
  2. 将变量的类型转换为 []byte，修改 字节数据
  3. 将修改后的字节数组 []byte 转换回 string
